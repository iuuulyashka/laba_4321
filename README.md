# Зайцева Юлия, 3 МО, РБД-1


# Основная логика:
Повар (1 поток) готовит наггетсы и раскладывает их по 3 тарелкам

Три толстяка (3 потока) едят наггетсы из своих тарелок

# Синхронизация: повар выложил → все толстяки поели → повар снова выложил

Условия завершения:

Кука уволили - хотя бы одна тарелка пустая ДО 5 дней

Кук не получил зарплату - все толстяки съели по 10000+ ДО 5 дней

Кук уволился сам - прошло 5 дней, но нет пустых тарелок и не все толстяки лопнули

# Ключевые особенности:

Строгая очередность через мьютекс и флаги синхронизации

1 день симуляции = 1 секунда реального времени

Автоматическая проверка достижения целевых сценариев

# Коэффициенты для сценариев:

Сценарий 1: efficiency=5, gluttony=150

Сценарий 2: efficiency=300, gluttony=250

Сценарий 3: efficiency=20, gluttony=15
